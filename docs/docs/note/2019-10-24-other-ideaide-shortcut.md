## 常用快捷键记录

| 功能             | 按键               | 前提/备注        |
| ---------------- | ------------------ | ---------------- |
| 查看定义         | ctrl+b             |
| 查看引用         | ctrl+b             | 光标在变量声明处 |
| 移动行           | ctrl+shift+up/down |
| 删除行           | ctrl+y             |
| 复制行           | ctrl+c             | 不选中任何       |
| 剪切行           | ctrl+x             |
| 关闭当前tab      | ctrl+F4            |
| extend selection | ctrl+w             |
| 格式化代码       | ctrl+alt+L         |
